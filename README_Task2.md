Customer Churn Insights Dashboard

📊 Project Overview
The Customer Churn Insights Dashboard is a data analytics project designed to analyze customer behavior, service usage patterns, and revenue trends. The dashboard provides meaningful insights that help identify why customers leave and how businesses can improve customer retention.
This project focuses on transforming raw customer data into interactive visual insights using Power BI.

🎯 Problem Statement
Customer churn is a major challenge for subscription-based businesses. Losing customers directly impacts revenue and growth. Businesses need to understand customer behavior, service usage, and billing patterns to reduce churn.
The objective of this project is to analyze customer data and identify patterns that influence customer churn, service preferences, and payment methods.

📁 Dataset Information

The dataset contains customer-related information including:

Customer ID

Contract Type

Internet Service

Payment Method

Monthly Charges

Total Charges

Customer Tenure

Churn Status

📌 Dashboard KPIs

The dashboard highlights key performance indicators such as:

Total Customers: 7,043

Average Monthly Charges: 64.76

Total Revenue: 16.06M

Churn Customers: 1.869K

📈 Key Insights
A significant number of customers have churned, highlighting retention challenges.

Customers using Fiber Optic internet services represent a large segment.

Electronic Check is the most commonly used payment method.

Customers without long-term contracts are more likely to churn.

Revenue growth is influenced by customer tenure and service usage.
